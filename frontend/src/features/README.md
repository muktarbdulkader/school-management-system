# Features Directory

This directory contains feature-based modules following the SRS requirements.

Each feature module should have:
- `components/` - React components specific to this feature
- `hooks/` - Custom hooks for this feature
- `services/` - API service functions for this feature

## Current Features:
- auth - Authentication & Authorization
- dashboard - Role-based dashboards
- students - Student management
- teachers - Teacher management
- parents - Parent management
- classes - Class & section management
- subjects - Subject management
- schedule - Schedule & timetable
- attendance - Attendance tracking
- assignments - Assignment management
- lessonPlans - Lesson planning
- objectives - Learning objectives
- communication - Chat, announcements, meetings
- leaveRequests - Leave management
- tasks - Task/todo management
- inventory - Materials & inventory
- library - Library management
- blog - School blog
- reports - Reports & analytics
- users - User management
- settings - System settings
